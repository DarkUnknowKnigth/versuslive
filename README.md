# Versuslive

Aplicacion del Taller de Aplicaciones serverless con Firebase

## Instalar
Run `npm install`

## Development server

Run `ng serve -o`

## Build

Run `ng build --prod`

## Subir a firebase

Run `ng build --prod`
Run `firebase login`
Run `firebase init`
Run `Cambiar el directorio (public) por dist/versuslive`
Run `firebase deploy`
